Instructions to run the program:

Select level of difficulty (easy / medium / hard). You can type it upper or lower case, it doesn't matter.


Easy mode: The player has to guess the right number and the game keeps going until the player guesses it correctly. The hints after each guess are “Too low” when the number is lower than the guess, and “Too high” when the number is higher than the guess. The participant has unlimited tries and at the end the number of attempts is given, as well as the recorded high score on easy mode and a replay option that he can use if selected.

Medium mode: The player has to guess the right number and the game keeps going until the player guesses it correctly. The hints after each guess are either “Super close!” when the distance between the guess and the correct number is equal or less than 4, “Close” when the distance between the guess and the correct number is equal or less than 10, and then “Far away” when neither of the past conditions are met. The participant has unlimited tries and at the end the number of attempts is given, as well as the recorded high score on medium mode and a replay option that he can use if selected.

Hard mode: This is basically the same as medium mode but the player has a limited number of tries (7 to be exact). When he passes 7 attempts the game is over and the recorded high score on hard mode is given as well as a replay option if selected.


At the end, either type Y/y to replay the game or N/n to quit the game. Highscores are saved depending on the game difficulty
